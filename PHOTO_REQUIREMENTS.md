All active website photos are in `assets/images/`. Current files are temporary copies of `hero.jpg`; replace each file in place with a real photo using the same filename.

## Replacement Rules

- Keep the exact filenames above.
- Use `.jpg` files for these photo slots.
- Replace files directly inside `assets/images/`.
- Do not rename icons in `assets/icons/`; those are separate UI assets.
- Avoid photos with visible customer addresses, license plates, private documents, or faces unless approved.
- Prefer real HM-PRO job photos over stock images.
